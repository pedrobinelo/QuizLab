## Project Description

<p align="justify">

**QuizLab** is a platform for creating quizzes that aims to modernize the teaching-learning process through active methodologies.

</p>

## Features

:heavy_check_mark: `Feature 1:` Register users so they can create and share quizzes.

:heavy_check_mark: `Feature 2:` Access quizzes using their unique code or link.

:heavy_check_mark: `Feature 3:` Create quizzes with a title, description, and cover image.

:heavy_check_mark: `Feature 4:` Create quiz questions that include a statement, supporting media (optional), and answer options.

:heavy_check_mark: `Feature 5:` Allow users to view, edit, and delete their created quizzes, as well as share them.

The project was built focusing on serverless performance and scalability:

- **Frontend:** Vanilla JavaScript (ES6+), HTML5, and CSS3.

- **Backend-as-a-Service:** Google Firebase (Authentication & Realtime Database).

- **Security:** Implementation of access rules (Security Rules) for data protection.
