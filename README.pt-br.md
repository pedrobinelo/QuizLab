## Descrição do projeto

<p align="justify">
 O **QuizLab** é uma plataforma para a criação de quizzes que visa modernizar o processo de ensino-
aprendizagem através de metodologias ativas.
</p>

## Funcionalidades

:heavy_check_mark: `Funcionalidade 1:` Cadastrar usuários a fim de que possam criar quizzes e compartilhá-los.

:heavy_check_mark: `Funcionalidade 2:` Acessar quizzes por meio de seu código único ou de seu link.

:heavy_check_mark: `Funcionalidade 3:` Criar quizzes com título, descrição e imagem de capa.

:heavy_check_mark: `Funcionalidade 4:` Criar questões para os quizzes que contenham enunciado, mídia de apoio (opcional) e opções de resposta.

:heavy_check_mark: `Funcionalidade 5:` Possibilitar ao usuário a visualização dos quizzes criados por ele, a edição e exclusão dos quizzes, além de seu compartilhamento.

## Ferramentas utilizadas

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
![Google Firebase](https://img.shields.io/badge/firebase-ffca28?style=for-the-badge&logo=firebase&logoColor=black)

O projeto foi construído focando em performance e escalabilidade serverless:

- **Frontend:** Vanilla JavaScript (ES6+), HTML5 e CSS3.
- **Backend-as-a-Service:** Google Firebase (Authentication & Realtime Database).
- **Segurança:** Implementação de regras de acesso (Security Rules) para proteção de dados.
